Division of labor

The backend AI owns the framework-neutral engine: codecs, worker bridges, bulk logic, build seams, service worker. This layer is in good shape and well-documented. It deliberately stopped before the UI.
The UI (this track) turns that engine into a usable app — starting with a runnable single-image compressor that keeps the current Squoosh look, then evolving the design gradually and adding the bulk interface without a disruptive redesign.
